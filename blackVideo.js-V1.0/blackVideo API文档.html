引入blackVideo.js后，实例化blackVideo构造函数

var blackVideo = new videoStart( video element , parent element ,[ video width , video height ]);

参数说明:

video element（必选参数）：video的DOM对象

parent element（必选参数）：播放器需要插入到这个父级元素的DOM对象后面

video width（可选参数）：播放器宽度 字符串类型、需要单位

video height（可选参数）：播放器高度 字符串类型、需要单位

* 您可以在video标签里指定播放器的宽高,也可以在构造函数里指定宽高,还可以使用我提供的可选方法来指定宽高.

例：下面例子往className盒子里面，添加一个宽500px高413px的播放器。

HTML 代码:
<div class="parent-box">
  <video id="myVideo" src="videoPath"><video>
</div>

JS 代码:
var parentBox = document.getElementsByClassName("parent-box")[0];
var myVideo = document.getElementById("myVideo");

//实例化blackVideo
var blackVideo = new videoStart( myVideo , parentBox , "500px" , "413px" );

如果实例化成功,您会在控制台看到一条语句. -- blackVideo.js starting success!

设置blackVideo的控制台样式

blackVideo.controlsStyle([ object ]);

此方法的为可选方法，可自定义控件的颜色

以键值对的形式，值为颜色值

可选属性如下：
progressIng: 进度条已播放的背景色
progressLoad: 进度条已缓冲的背景色
progressBar: 进度条总长度的背景色
progressBtn: 进度条按钮的背景色
videoFiller: 视频补白背景色
videoHeaderBGColor: 视频标题背景色
videoHeaderFontColor: 视频标题字体色
videoHeaderFontSize: 视频标题字体大小
controlsBGColor: 控制台底部背景色
volumeIng: 音量进度条背景色
volumeBar: 音量总进度条背景色
volumeBtn: 音量拖动按钮背景色
playBtn: 播放按钮背景色
currentColor: 当前播放时长背景色
durationColor: 总播放时长背景色
bisaColor: 播放时长与总时长中间的分割线的颜色
hornColor: 喇叭按钮颜色
downloadBGColor: 下载按钮框的颜色
downloadBtnColor: 下载按钮的颜色

例：
blackVideo.controlsStyle({
  progressIng : "#000",
  progressLoad : "rgb(0,0,0)",
  progressBar : "black"
});

启动控制台

blackVideo.controlsStart([ isDownloadVideo , videoHeader ]);

参数说明：

isDownloadVideo（可选参数）：是否下载视频，默认为 true

videoHeader（可选参数）：播放器标题，默认为最后一个 / 后面的所有字符串

例：
blackVideo.controlsStart( true , "视频标题" );

其它可选方法

blackVideo.setWidth(); 设置播放器宽度 传入带有单位的值
blackVideo.setHeight();设置播放器高度 传入带有单位的值
blackVideo.videoPlay();开始播放
blackVideo.videoPause();暂停播放

下面是实例:使用默认UI控件样式的播放器


用了几行js代码就完成了一个简洁的播放器,是不是很简单?

代码如下:

<html>
  <head>
    <meta charset="utf-8" />
    <title></title>
    <link rel="stylesheet" href="./css/fontello.css" type="text/css"/>  //引入控件图标
    <link rel="stylesheet" href="./css/blackVideo.css" type="text/css"/>  //引入blackVideo的基本css样式
  </head>
  <body>
    <div class="example-box">
      <div class="parent-box"></div>
      <video src="./video.mp4" id="myVideo-one"></video>
    </div>
  </body>
  <script src="./blackVideo.js"></script>
  <script type="text/javascript">
    <!-- 脚本部分 -->
    var parentBox = document.getElementsByClassName("parent-box")[0];
    var myVideoOne = document.getElementById("myVideo-one");
    var blackVideo = new videoStart(myVideoOne,parentBox,"700px","431px");
    blackVideo.controlsStart();
  </script>
</html>
